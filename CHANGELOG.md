# سجل التغييرات

جميع التغييرات المهمة في هذا المشروع سيتم توثيقها في هذا الملف.

## [1.0.0] - 2024-11-08

### المضافة
- واجهة مستخدم رسومية حديثة مع ثيم داكن
- دعم 5 أنظمة رومنة مختلفة:
  - النظام الحالي (القواعد الأساسية)
  - ALA-LC (مكتبة الكونغرس الأمريكية)
  - DMG (الجمعية الألمانية للدراسات الشرقية)
  - ISO 233 (المعيار الدولي)
  - IJMES (المجلة الدولية لدراسات الشرق الأوسط)
- استيراد ملفات TXT و DOCX
- تصدير النتائج إلى ملفات نصية
- نسخ النتائج إلى الحافظة
- تحويل تلقائي عند تغيير النظام
- دعم النصوص المشكّلة وغير المشكّلة
- واجهة عربية بالكامل مع دعم RTL

### التقنية
- Python 3.7+
- PyQt6 للواجهة الرسومية
- python-docx لدعم ملفات Word (اختياري)
